## Getting Started

To run the application locally:
```bash
npm install
npm run dev
```

The application is hosted at `http://localhost:3000`.

## Key Features
- **Habit Tracking**: 7-day rolling view and monthly grid.
- **Routine Planning**: Morning, afternoon, and evening blocks.
- **Drag-and-Drop**: Interactive reordering for all items.
- **PDF Export**: Print-optimized layout for trackable sheets.
